Cookies и локальные настройкиCookies and local preferences

Здесь описано, что сайт сохраняет в браузере и какие технологии сейчас не используются.This page explains what the website stores in the browser and which technologies are not currently used.

1. Что используется сейчас1. What is currently used

Сайт не использует рекламные cookies, системы веб-аналитики или стороннее поведенческое отслеживание. Для интерфейса применяются только значения в localStorage браузера.The website does not use advertising cookies, web analytics or third-party behavioural tracking. Only browser localStorage values are used for the interface.

2. Какие значения сохраняются2. Stored values

  • mgw_lang — сохраняет выбранный язык текущей версии сайта;stores the selected language in the current website version;
  • mgw_cookie_choice — сохраняет факт закрытия информационного окна, чтобы не показывать его при каждом открытии.stores that the information window was closed so it is not shown on every visit.

Эти значения не используются для создания рекламного профиля и не передаются рекламным системам.These values are not used to build an advertising profile and are not sent to advertising systems.

3. Что означают кнопки уведомления3. What the notice buttons mean

Поскольку необязательные технологии сейчас не подключены, ни одна кнопка уведомления не запускает аналитику, рекламу или маркетинговые cookies. Нажатие только сохраняет выбранное значение и закрывает окно.Because optional technologies are not connected, neither notice button starts analytics, advertising or marketing cookies. A click only stores the selected value and closes the window.

4. Необязательные технологии в будущем4. Optional technologies in the future

Если позже будут подключены аналитика, маркетинговые инструменты или другие необязательные технологии, они не должны запускаться до получения соответствующего выбора пользователя. Политика и элементы управления будут обновлены до такого подключения.If analytics, marketing tools or other optional technologies are added later, they must not run before the user makes the relevant choice. This policy and the controls will be updated before such tools are connected.

5. Как удалить настройки5. How to remove preferences

Удалите данные сайта в настройках браузера. После удаления язык вернётся к значению по умолчанию, а информационное окно появится снова. Ссылка «Настройки cookies» в подвале главной страницы также повторно открывает окно.Remove the website data in your browser settings. The language will return to its default and the information window will appear again. The Cookie settings link in the main page footer also reopens the window.

6. Telegram Mini App6. Telegram Mini App

Игровой профиль, матчи, балансы и заявки хранятся на сервере проекта и описаны в Политике конфиденциальности. Они не являются cookie-настройками сайта.Game profiles, matches, balances and requests are stored on the project server and described in the Privacy Policy. They are not website cookie preferences.

7. Обновление политики7. Policy updates

Последнее обновление: 10 июля 2026 года. Политика будет пересматриваться при изменении технического состава сайта или переходе на отдельные языковые URL.Last updated: 10 July 2026. This policy will be reviewed when the website’s technical setup changes or separate language URLs are introduced.
