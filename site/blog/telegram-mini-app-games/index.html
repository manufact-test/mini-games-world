Что такое Telegram Mini AppsWhat are Telegram Mini Apps

Почему игра открывается внутри Telegram без отдельной установки и как разделены бот, Mini App и сервер.Why the game opens inside Telegram without installation and how the bot, Mini App and server are separated.

10.07.20267 min

Telegram Mini App — это веб-приложение, которое открывается внутри Telegram. Пользователь нажимает кнопку в боте и получает полноценный игровой интерфейс без перехода в магазин приложений.

Как происходит запуск

Бот @MiniGamesWorld_bot отправляет кнопку «Открыть игру». Telegram запускает Mini App и передаёт подписанные данные пользователя. Сервер проверяет их и создаёт или обновляет игровой профиль.

Что делает Telegram-бот сейчас

Бот является точкой входа в проект: команды /start, /app и /help показывают сообщение с кнопкой запуска Mini App. Пользовательские игровые приглашения и автоматические уведомления о матчах пока не подключены.

Что делает Mini App

В Mini App доступны профиль, отдельные балансы Match и Gold, крестики-нолики, выбор комнаты и поля, поиск соперника, игровое поле, история матчей, история баланса, заявки на пополнение, магазин призов и формы обращения в поддержку.

Чем Mini App отличается от обычного сайта

Обычный сайт не получает подтверждённый контекст Telegram автоматически. Mini App открывается из Telegram и передаёт серверу подписанные данные запуска. Благодаря этому пользователю не нужно создавать ещё один логин и пароль.

Почему не нужна установка

Интерфейс загружается с сервера как веб-приложение. После публикации обновлённые файлы становятся доступны без скачивания новой версии из магазина приложений.

Как разделена система

  • Telegram-бот открывает Mini App.
  • Mini App показывает интерфейс и отправляет действия игрока.
  • Сервер проверяет пользователя, матчи, ходы, таймеры, балансы, историю и заявки.

Критичные данные не должны зависеть только от открытого окна Telegram. Завершённые матчи и операции сохраняются на сервере.

Телефон и компьютер

Mini App работает в поддерживаемых версиях Telegram на смартфонах и компьютерах. Интерфейс Mini Games World сделан mobile-first, но адаптируется под более широкий экран.

Один игровой профиль должен использоваться через Telegram. Открытие Mini App как обычного сайта на публичном домене не создаёт полноценного пользователя и завершается просьбой открыть приложение через Telegram.

Какие функции появятся позже

В следующих этапах планируются приглашения конкретным друзьям, дополнительные уведомления, новые игры и дальнейшее развитие Gold-комнаты. На сайте эти функции отмечаются как будущие, пока они не подключены в рабочем приложении.

A Telegram Mini App is a web application that opens inside Telegram. A user taps a bot button and receives a complete game interface without visiting an app store.

How launch works

@MiniGamesWorld_bot sends an Open Game button. Telegram launches the Mini App and supplies signed user data. The server validates that data and creates or updates the game profile.

What the bot currently does

The bot is the project entry point. The /start, /app and /help commands show a message with the Mini App button. Player invitations and automatic match notifications are not connected yet.

What the Mini App currently does

The Mini App provides a profile, separate Match and Gold balances, tic-tac-toe, room and board selection, matchmaking, the game board, match history, balance history, top-up requests, the prize shop and support forms.

How it differs from a normal website

A normal website does not automatically receive verified Telegram context. A Mini App launches from Telegram and sends signed launch data to the server, so another username and password are not required.

Why no installation is required

The interface loads from the server as a web application. Published updates become available without downloading a new app-store package.

System responsibilities

  • The Telegram bot opens the Mini App.
  • The Mini App displays the interface and sends player actions.
  • The server validates users, matches, moves, timers, balances, history and requests.

Critical data is stored on the server rather than only in the open Telegram window.

Mobile and desktop

The Mini App works in supported Telegram versions on phones and computers. Mini Games World is mobile-first and also adapts to wider screens.

The game profile is intended to be used through Telegram. Opening the public Mini App URL as a normal website does not create a full player and asks the visitor to open it through Telegram.

Future features

Direct friend invitations, additional notifications, new games and further Gold development are planned. They are described as future features until they are connected in the working product.
