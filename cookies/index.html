Здесь описано, какие данные сайт сохраняет в браузере и как управлять своим выбором.This page explains what the website stores in your browser and how to manage your choice.

1. Что используется сейчас1. What is currently used

На текущем этапе сайт не использует рекламные cookies и сторонние системы поведенческого отслеживания. Для работы интерфейса применяются только локальные настройки браузера.At the current stage, the website does not use advertising cookies or third-party behavioural tracking. Only local browser preferences are used for interface functionality.

2. Необходимые настройки2. Necessary preferences

  • mgw_lang — сохраняет выбранный язык сайта;stores the selected website language;
  • mgw_cookie_choice — сохраняет ваш выбор в cookie-баннере, чтобы не показывать его при каждом открытии.stores your banner choice so it is not shown on every visit.

Эти значения хранятся в localStorage браузера и необходимы для запрошенной вами функциональности интерфейса.These values are stored in browser localStorage and are necessary for the interface functionality you requested.

3. Необязательные технологии3. Optional technologies

Если в будущем будут подключены аналитика, маркетинговые инструменты или другие необязательные технологии, они не должны запускаться до получения соответствующего согласия. Информация на этой странице и настройки баннера будут обновлены.If analytics, marketing tools or other optional technologies are added later, they should not run before the relevant consent is obtained. This page and the banner controls will be updated accordingly.

4. Как изменить выбор4. How to change your choice

Вы можете удалить данные сайта в настройках браузера. После этого cookie-баннер появится снова. Также можно использовать ссылку «Настройки cookies» в подвале главной страницы.You can remove the website data in your browser settings. The banner will then appear again. You may also use the “Cookie settings” link in the main page footer.

5. Основа выбора5. Basis for choice

Для строго необходимых технологий отдельное согласие обычно не требуется. Необязательные технологии требуют понятного предварительного выбора, а отказ должен быть не сложнее принятия.Strictly necessary technologies generally do not require separate consent. Optional technologies require a clear prior choice, and rejecting them should be no harder than accepting them.
