# Каталог призов Mini Games World

Каталог магазина хранится в `prizes.php`. Для изменения ассортимента не нужно редактировать `ShopService` или интерфейс приложения.

## Структура позиции

- `id` — постоянный уникальный ID позиции;
- `country_code` — код страны из массива `countries`;
- `provider_code` — технический код провайдера;
- `provider` — отображаемое название;
- `title` и `description` — текст карточки;
- `image` и `image_alt` — изображение и alt-текст;
- `delivery_type` — способ выдачи;
- `sort_order` — порядок отображения;
- `enabled` — доступность позиции;
- `denominations` — доступные номиналы и их стоимость в Gold.

## Временное отключение

Чтобы скрыть страну, позицию или номинал, установите `enabled => false`. Удалять запись не нужно: постоянные ID пригодятся для истории старых заказов.

## Важные правила

- ID позиций и номиналов не должны повторяться.
- После появления реальных заказов нельзя переиспользовать старый ID для другого приза.
- Стоимость указывается в поле `gold_cost` целым положительным числом.
- Изображения будут подключены к витрине в MVP-8.2.
